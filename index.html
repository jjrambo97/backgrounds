Multiple backgrounds for the site plug.dj.

*I do not own the backgrounds nor did I make them

To install:

  1. Select a background you want

  2. Click the download button

  3. Drag the downloaded file on your extensions page

  4. A popup should appear, click "Add"

  5. Refresh Plug

You can install all of the backgrounds if you want and switch them out by enabling only one.
